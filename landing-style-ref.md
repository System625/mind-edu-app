# Landing Page Style Reference

This document contains visual references for the "Premium" structural design of the Mind-Edu app, based on the high-impact layouts and typography of the "Superpower" design.

![Superpower Reference Image](file:///C:/Users/Olatheruler/.gemini/antigravity/brain/e98f4cdf-225f-42ef-8aee-ba6ee276464b/media__1772503596997.jpg)

### Key Structural Notes
- **Typography**: Ultra-bold, large-scale heroes with tight tracking.
- **Spacing**: Deep vertical padding between sections (py-40).
- **Layering**: Clean white space with subtle glassmorphism overlays.
- **Pill Elements**: Fully rounded buttons and badges.
